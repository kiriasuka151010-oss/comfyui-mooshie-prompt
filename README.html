comfyui-mooshie-prompt

Mooshie 画师浏览 + Danbooru 标签搜索 + 可编辑 Prompt — 一根线搞定

🧩 节点

  • MooshieBrowser — 上半浏览 42k+ Mooshie 画师，下半搜索 D站 画廊，瀑布流展示
  • EditablePrompt — 6 类标签（质量/画师/角色/系列/常规/元），每类独立跟随/锁定开关

🚀 快速开始

  1. 安装到 ComfyUI custom_nodes/ 目录
  2. 重启 ComfyUI
  3. 拖入 workflows/smoke-test.json（基础测试）或 workflows/baselinelora.json（带 LoRA）
  4. 在 MooshieBrowser 搜索 D站 或浏览画师，选中 → EditablePrompt 自动填入
  5. 调整锁定/跟随后点 Queue

📋 工作流

smoke-test.json — 基础

MooshieBrowser → EditablePrompt → CLIPTextEncode → KSampler → VAEDecode → SaveImage
  • 无 LoRA 纯验证插件功能
  • MooshieBrowser 选图后 EditablePrompt 自动填入 6 类标签
  • 每类可切换跟随/锁定，清空跟随恢复上游值

baselinelora.json — LoRA 生产

MooshieBrowser → EditablePrompt → CLIPTextEncode → Power LoRA Loader → KSampler → VAEDecode → SaveImage
  • anima_baseV10 + Power LoRA Loader（rgthree）
  • 预装了用户调好的 26 个 Anima LoRA，turbo 已开启
  • 768×1536 竖版，7 步 euler，CFG 0.7

✨ 特性

  • 🎨 Mooshie 画师浏览 — CDN 缩略图预览，搜索/排序 42k+ AI 风格画师
  • 📋 D站 标签搜索 — 中文/英文模糊 tag 联想，点击填入
  • 🏷️ 5 类原生标签 — 画师 / 角色 / 系列 / 常规 / 元，Danbooru 官方分类
  • 🔒 逐类跟随/锁定 — 每类独立开关，跟随=同步上游，锁定=手动改
  • ❤️ 收藏系统 — 保存喜欢的 D站 图，一键复用
  • 📱 瀑布流画廊 — 原比例展示，竖图高横图矮
  • 🖼️ 双击大图 — 原图+下载
  • 🔞 分级过滤 — S/Q/E 三键切换
  • 🔗 单线连接 — MooshieBrowser 只输出一根 tag_data，EditablePrompt 只接一根 upstream_data

📦 依赖

  • ComfyUI
  • 可选 Power LoRA Loader（rgthree-comfy）— baselinelora.json 需要
  • 可选 PreviewAny（comfy-core）— prompt 预览

🔧 配置

首次启动自动从原版 ComfyUI-Danbooru-Anima-Prompt 加载中文标签索引（50k+ 条）。无需额外配置。
